   <p>In <a href="/wiki/Mathematics" title="Mathematics">mathematics</a>, the <b>natural numbers</b> are those used for <a href="/wiki/Counting" title="Counting">counting</a> (as in "there are <i>six</i> coins on the table") and <a href="/wiki/Total_order" title="Total order">ordering</a> (as in "this is the <i>third</i> largest city in the country"). In common mathematical terminology, words colloquially used for counting are "<a href="/wiki/Cardinal_number_(linguistics)" title="Cardinal number (linguistics)">cardinal numbers</a>" and words connected to ordering represent "<a href="/wiki/Ordinal_number_(linguistics)" title="Ordinal number (linguistics)">ordinal numbers</a>". The natural numbers can, at times, appear as a convenient set of codes (labels or "names"); that is, as what <a href="/wiki/Linguists" class="mw-redirect" title="Linguists">linguists</a> call <a href="/wiki/Nominal_number" title="Nominal number">nominal numbers</a>, foregoing many or all of the properties of being a number in a mathematical sense. </p>
   <p>Some definitions, including the standard <a href="/wiki/ISO_80000-2" title="ISO 80000-2">ISO 80000-2</a>,<sup id="cite_ref-ISO80000_1-0" class="reference"><a href="#cite_note-ISO80000-1">[1]</a></sup> begin the natural numbers with <a href="/wiki/0_(number)" class="mw-redirect" title="0 (number)">0</a>, corresponding to the <b>non-negative integers</b> <span class="texhtml">0, 1, 2, 3, …</span>, whereas others start with 1, corresponding to the <b>positive integers</b> <span class="texhtml"><a href="/wiki/1_(number)" class="mw-redirect" title="1 (number)">1</a>, <a href="/wiki/2_(number)" class="mw-redirect" title="2 (number)">2</a>, <a href="/wiki/3_(number)" class="mw-redirect" title="3 (number)">3</a>, …</span>.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup><sup id="cite_ref-3" class="reference"><a href="#cite_note-3">[3]</a></sup><sup id="cite_ref-4" class="reference"><a href="#cite_note-4">[4]</a></sup><sup id="cite_ref-Mac_Lane_&amp;_Birkhoff_1999_p15_5-0" class="reference"><a href="#cite_note-Mac_Lane_&amp;_Birkhoff_1999_p15-5">[5]</a></sup> Texts that exclude zero from the natural numbers sometimes refer to the natural numbers together with zero as the <b>whole numbers</b>, but in other writings, that term is used instead for the integers (including negative integers).<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">[6]</a></sup> </p>
   <p>The natural numbers are a basis from which many other number sets may be built by extension: the <a href="/wiki/Integer" title="Integer">integers</a> (<a href="/wiki/Grothendieck_group" title="Grothendieck group">Grothendieck group</a>), by including (if not yet in) the <a href="/wiki/Neutral_element" class="mw-redirect" title="Neutral element">neutral element</a> 0 and an <a href="/wiki/Additive_inverse" title="Additive inverse">additive inverse</a> (−<i>n</i>) for each nonzero natural number <i>n</i>; the <a href="/wiki/Rational_number" title="Rational number">rational numbers</a>, by including a <a href="/wiki/Multiplicative_inverse" title="Multiplicative inverse">multiplicative inverse</a> (1/<i>n</i>) for each nonzero integer <i>n</i> (and also the product of these inverses by integers); the <a href="/wiki/Real_number" title="Real number">real numbers</a> by including with the rationals the <a href="/wiki/Limit_(mathematics)" title="Limit (mathematics)">limits</a> of (converging) <a href="/wiki/Construction_of_the_real_numbers#Construction_from_Cauchy_sequences" title="Construction of the real numbers">Cauchy sequences</a> of rationals; the <a href="/wiki/Complex_number" title="Complex number">complex numbers</a>, by including with the real numbers the unresolved <a href="/wiki/Imaginary_unit" title="Imaginary unit">square root of minus one</a> (and also the sums and products thereof); and so on.<sup id="cite_ref-7" class="reference"><a href="#cite_note-7">[7]</a></sup><sup id="cite_ref-8" class="reference"><a href="#cite_note-8">[8]</a></sup> These chains of extensions make the natural numbers canonically <a href="/wiki/Embedding" title="Embedding">embedded</a> (identified) in the other number systems. </p>
   <p>Properties of the natural numbers, such as <a href="/wiki/Divisibility" class="mw-redirect" title="Divisibility">divisibility</a> and the distribution of <a href="/wiki/Prime_number" title="Prime number">prime numbers</a>, are studied in <a href="/wiki/Number_theory" title="Number theory">number theory</a>. Problems concerning counting and ordering, such as <a href="/wiki/Partition_(number_theory)" title="Partition (number theory)">partitioning</a> and <a href="/wiki/Enumerative_combinatorics" title="Enumerative combinatorics">enumerations</a>, are studied in <a href="/wiki/Combinatorics" title="Combinatorics">combinatorics</a>. </p>
   <p>In common language, for example in <a href="/wiki/Primary_school" title="Primary school">primary school</a>, natural numbers may be called <b>counting numbers</b><sup id="cite_ref-MathWorld_CountingNumber_9-0" class="reference"><a href="#cite_note-MathWorld_CountingNumber-9">[9]</a></sup> both to intuitively exclude the negative integers and zero, and also to contrast the <a href="/wiki/Discrete_mathematics" title="Discrete mathematics">discreteness</a> of <a href="/wiki/Counting" title="Counting">counting</a> to the <a href="/wiki/List_of_continuity-related_mathematical_topics" title="List of continuity-related mathematical topics">continuity</a> of <a href="/wiki/Measurement" title="Measurement">measurement</a>, established by the <a href="/wiki/Real_number" title="Real number">real numbers</a>. </p>

   <h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Natural_number&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
   <h3><span class="mw-headline" id="Ancient_roots">Ancient roots</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Natural_number&amp;action=edit&amp;section=2" title="Edit section: Ancient roots">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
   <div class="thumb tright">
    <div class="thumbinner" style="width:172px;">
     <a href="/wiki/File:Os_d%27Ishango_IRSNB.JPG" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/42/Os_d%27Ishango_IRSNB.JPG/170px-Os_d%27Ishango_IRSNB.JPG" decoding="async" width="170" height="275" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/42/Os_d%27Ishango_IRSNB.JPG/255px-Os_d%27Ishango_IRSNB.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/42/Os_d%27Ishango_IRSNB.JPG/340px-Os_d%27Ishango_IRSNB.JPG 2x" data-file-width="1404" data-file-height="2274"></a>

   <p>The most primitive method of representing a natural number is to put down a mark for each object. Later, a set of objects could be tested for equality, excess or shortage, by striking out a mark and removing an object from the set. </p>
   <p>The first major advance in abstraction was the use of <a href="/wiki/Numeral_system" title="Numeral system">numerals</a> to represent numbers. This allowed systems to be developed for recording large numbers. The ancient <a href="/wiki/History_of_Ancient_Egypt" class="mw-redirect" title="History of Ancient Egypt">Egyptians</a> developed a powerful system of numerals with distinct <a href="/wiki/Egyptian_hieroglyphs" title="Egyptian hieroglyphs">hieroglyphs</a> for&nbsp;1, 10, and all the powers of&nbsp;10 up to over 1&nbsp;million. A stone carving from <a href="/wiki/Karnak" title="Karnak">Karnak</a>, dating from around 1500&nbsp;BC and now at the <a href="/wiki/Louvre" title="Louvre">Louvre</a> in Paris, depicts&nbsp;276 as 2&nbsp;hundreds, 7&nbsp;tens, and 6&nbsp;ones; and similarly for the number&nbsp;4,622. The <a href="/wiki/Babylonia" title="Babylonia">Babylonians</a> had a <a href="/wiki/Positional_notation" title="Positional notation">place-value</a> system based essentially on the numerals for&nbsp;1 and&nbsp;10, using base sixty, so that the symbol for sixty was the same as the symbol for one, its value being determined from context.<sup id="cite_ref-13" class="reference"><a href="#cite_note-13">[13]</a></sup> </p>
   <p>A much later advance was the development of the idea that&nbsp;<a href="/wiki/0_(number)" class="mw-redirect" title="0 (number)">0</a> can be considered as a number, with its own numeral. The use of a&nbsp;0 <a href="/wiki/Numerical_digit" title="Numerical digit">digit</a> in place-value notation (within other numbers) dates back as early as 700&nbsp;BC by the Babylonians, but they omitted such a digit when it would have been the last symbol in the number.<sup id="cite_ref-14" class="reference"><a href="#cite_note-14">[14]</a></sup> The <a href="/wiki/Olmec" class="mw-redirect" title="Olmec">Olmec</a> and <a href="/wiki/Maya_civilization" title="Maya civilization">Maya civilizations</a> used&nbsp;0 as a separate number as early as the <span class="nowrap">1st century BC</span>, but this usage did not spread beyond <a href="/wiki/Mesoamerica" title="Mesoamerica">Mesoamerica</a>.<sup id="cite_ref-15" class="reference"><a href="#cite_note-15">[15]</a></sup><sup id="cite_ref-16" class="reference"><a href="#cite_note-16">[16]</a></sup> The use of a numeral&nbsp;0 in modern times originated with the <a href="/wiki/India" title="India">Indian</a> mathematician <a href="/wiki/Brahmagupta" title="Brahmagupta">Brahmagupta</a> in 628. However, 0 had been used as a number in the medieval <a href="/wiki/Computus" title="Computus">computus</a> (the calculation of the date of <a href="/wiki/Easter" title="Easter">Easter</a>), beginning with <a href="/wiki/Dionysius_Exiguus" title="Dionysius Exiguus">Dionysius Exiguus</a> in 525, without being denoted by a numeral (standard <a href="/wiki/Roman_numerals" title="Roman numerals">Roman numerals</a> do not have a symbol for&nbsp;0); instead <i>nulla</i> (or the genitive form <i>nullae</i>) from <i>nullus</i>, the Latin word for "none", was employed to denote a&nbsp;0 value.<sup id="cite_ref-17" class="reference"><a href="#cite_note-17">[17]</a></sup> </p>
   <p>The first systematic study of numbers as <a href="/wiki/Abstraction" title="Abstraction">abstractions</a> is usually credited to the <a href="/wiki/Ancient_Greece" title="Ancient Greece">Greek</a> philosophers <a href="/wiki/Pythagoras" title="Pythagoras">Pythagoras</a> and <a href="/wiki/Archimedes" title="Archimedes">Archimedes</a>. Some Greek mathematicians treated the number&nbsp;1 differently than larger numbers, sometimes even not as a number at all.<sup id="cite_ref-18" class="reference"><a href="#cite_note-18">[18]</a></sup> </p>
   <p>Independent studies also occurred at around the same time in <a href="/wiki/India" title="India">India</a>, <a href="/wiki/China" title="China">China</a>, and <a href="/wiki/Mesoamerica" title="Mesoamerica">Mesoamerica</a>.<sup id="cite_ref-19" class="reference"><a href="#cite_note-19">[19]</a></sup> </p> 
   <h3><span class="mw-headline" id="Modern_definitions">Modern definitions</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Natural_number&amp;action=edit&amp;section=3" title="Edit section: Modern definitions">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
   <p>In <a href="/wiki/19th_century" title="19th century">19th century</a> <a href="/wiki/Europe" title="Europe">Europe</a>, there was mathematical and philosophical discussion about the exact nature of the natural numbers. A school<sup class="noprint Inline-Template" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Avoid_weasel_words" class="mw-redirect" title="Wikipedia:Avoid weasel words"><span title="The material near this tag possibly uses too vague attribution or weasel words. (March 2017)">which?</span></a></i>]</sup> of <a href="/wiki/Naturalism_(philosophy)" title="Naturalism (philosophy)">Naturalism</a> stated that the natural numbers were a direct consequence of the human psyche. <a href="/wiki/Henri_Poincar%C3%A9" title="Henri Poincaré">Henri Poincaré</a> was one of its advocates, as was <a href="/wiki/Leopold_Kronecker" title="Leopold Kronecker">Leopold Kronecker</a> who summarized "God made the integers, all else is the work of man".<sup id="cite_ref-20" class="reference"><a href="#cite_note-20">[20]</a></sup> </p>
   <p>In opposition to the Naturalists, the <a href="/wiki/Constructivism_(mathematics)" class="mw-redirect" title="Constructivism (mathematics)">constructivists</a> saw a need to improve the logical rigor in the <a href="/wiki/Foundations_of_mathematics" title="Foundations of mathematics">foundations of mathematics</a>.<sup id="cite_ref-21" class="reference"><a href="#cite_note-21">[21]</a></sup> In the 1860s, <a href="/wiki/Hermann_Grassmann" title="Hermann Grassmann">Hermann Grassmann</a> suggested a recursive definition for natural numbers thus stating they were not really natural but a consequence of definitions. Later, two classes of such formal definitions were constructed; later, they were shown to be equivalent in most practical applications. </p>
   <p><a href="/wiki/Set-theoretical_definitions_of_natural_numbers" class="mw-redirect" title="Set-theoretical definitions of natural numbers">Set-theoretical definitions of natural numbers</a> were initiated by <a href="/wiki/Frege" class="mw-redirect" title="Frege">Frege</a> and he initially defined a natural number as the class of all sets that are in one-to-one correspondence with a particular set, but this definition turned out to lead to paradoxes including <a href="/wiki/Russell%27s_paradox" title="Russell's paradox">Russell's paradox</a>. Therefore, this formalism was modified so that a natural number is defined as a particular set, and any set that can be put into one-to-one correspondence with that set is said to have that number of elements.<sup id="cite_ref-22" class="reference"><a href="#cite_note-22">[22]</a></sup> </p>
   <p>The second class of definitions was introduced by <a href="/wiki/Charles_Sanders_Peirce" title="Charles Sanders Peirce">Charles Sanders Peirce</a>, refined by <a href="/wiki/Richard_Dedekind" title="Richard Dedekind">Richard Dedekind</a>, and further explored by <a href="/wiki/Giuseppe_Peano" title="Giuseppe Peano">Giuseppe Peano</a>; this approach is now called <a href="/wiki/Peano_arithmetic" class="mw-redirect" title="Peano arithmetic">Peano arithmetic</a>. It is based on an <a href="/wiki/Axiomatization" class="mw-redirect" title="Axiomatization">axiomatization</a> of the properties of <a href="/wiki/Ordinal_number" title="Ordinal number">ordinal numbers</a>: each natural number has a successor and every non-zero natural number has a unique predecessor. Peano arithmetic is <a href="/wiki/Equiconsistent" class="mw-redirect" title="Equiconsistent">equiconsistent</a> with several weak systems of set theory. One such system is <a href="/wiki/ZFC" class="mw-redirect" title="ZFC">ZFC</a> with the <a href="/wiki/Axiom_of_infinity" title="Axiom of infinity">axiom of infinity</a> replaced by its negation. Theorems that can be proved in ZFC but cannot be proved using the Peano Axioms include <a href="/wiki/Goodstein%27s_theorem" title="Goodstein's theorem">Goodstein's theorem</a>.<sup id="cite_ref-23" class="reference"><a href="#cite_note-23">[23]</a></sup> </p>
   <p>With all these definitions it is convenient to include&nbsp;0 (corresponding to the <a href="/wiki/Empty_set" title="Empty set">empty set</a>) as a natural number. Including&nbsp;0 is now the common convention among <a href="/wiki/Set_theory" title="Set theory">set theorists</a><sup id="cite_ref-24" class="reference"><a href="#cite_note-24">[24]</a></sup> and <a href="/wiki/Logic" title="Logic">logicians</a>.<sup id="cite_ref-25" class="reference"><a href="#cite_note-25">[25]</a></sup> Other mathematicians also include&nbsp;0<sup id="cite_ref-Mac_Lane_&amp;_Birkhoff_1999_p15_5-1" class="reference"><a href="#cite_note-Mac_Lane_&amp;_Birkhoff_1999_p15-5">[5]</a></sup>, e.g., <a href="/wiki/Computer_language" title="Computer language">Computer languages</a> often <a href="/wiki/Zero-based_numbering" title="Zero-based numbering">start from zero</a> when enumerating items like <a href="/wiki/For_loop" title="For loop">loop counters</a> and <a href="/wiki/String_(computer_science)" title="String (computer science)">string-</a> or <a href="/wiki/Array_data_structure" title="Array data structure">array-elements</a>.<sup id="cite_ref-26" class="reference"><a href="#cite_note-26">[26]</a></sup><sup id="cite_ref-27" class="reference"><a href="#cite_note-27">[27]</a></sup> Many mathematicians have kept the older tradition and take&nbsp;1 to be the first natural number.<sup id="cite_ref-28" class="reference"><a href="#cite_note-28">[28]</a></sup> </p>
   <p>Since different properties are customarily associated to the tokens <span class="texhtml">0</span> and <span class="texhtml">1</span>, e.g., neutral elements for addition and multiplications, respectively, it is important to know which version of <i>natural numbers</i>, generically denoted by <span class="mwe-math-element"><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;">

   <h2><span class="mw-headline" id="Properties">Properties</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Natural_number&amp;action=edit&amp;section=5" title="Edit section: Properties">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
   <h3><span class="mw-headline" id="Infinity">Infinity</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Natural_number&amp;action=edit&amp;section=6" title="Edit section: Infinity">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
   <p>The set of natural numbers is an <a href="/wiki/Infinite_set" title="Infinite set">infinite set</a>. This kind of <a href="/wiki/Infinity" title="Infinity">infinity</a> is, by definition, called <a href="/wiki/Countably_infinite" class="mw-redirect" title="Countably infinite">countable infinity</a>. All sets that can be put into a <a href="/wiki/Bijective" class="mw-redirect" title="Bijective">bijective</a> relation to the natural numbers are said to have this kind of infinity. This is also expressed by saying that the <a href="/wiki/Cardinal_number" title="Cardinal number">cardinal number</a> of the set is <a href="/wiki/Aleph_number#Aleph-naught" title="Aleph number">aleph-naught</a> (<span class="texhtml">ℵ<sub>0</sub></span>).<sup id="cite_ref-32" class="reference"><a href="#cite_note-32">[32]</a></sup> </p> 
   <h3><span class="mw-headline" id="Addition">Addition</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Natural_number&amp;action=edit&amp;section=7" title="Edit section: Addition">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
   <p>One can recursively define an <a href="/wiki/Addition_in_N" class="mw-redirect" title="Addition in N">addition</a> <a href="/wiki/Operation_(mathematics)" title="Operation (mathematics)">operator</a> on the natural numbers by setting <span class="texhtml"><i>a</i> + 0 = <i>a</i></span> and <span class="texhtml"><i>a</i> + <i>S</i>(<i>b</i>) = <i>S</i>(<i>a</i> + <i>b</i>)</span> for all <span class="texhtml"><i>a</i></span>, <span class="texhtml"><i>b</i></span>. Here <span class="texhtml"><i>S</i></span> should be read as "<a href="/wiki/Successor_function" title="Successor function">successor</a>". This turns the natural numbers <span class="texhtml">(ℕ, +)</span> into a <a href="/wiki/Commutative" class="mw-redirect" title="Commutative">commutative</a> <a href="/wiki/Monoid" title="Monoid">monoid</a> with <a href="/wiki/Identity_element" title="Identity element">identity element</a>&nbsp;0, the so-called <a href="/wiki/Free_object" title="Free object">free object</a> with one generator. This monoid satisfies the <a href="/wiki/Cancellation_property" title="Cancellation property">cancellation property</a> and can be embedded in a <a href="/wiki/Group_(mathematics)" title="Group (mathematics)">group</a> (in the mathematical sense of the word <i>group</i>). The smallest group containing the natural numbers is the <a href="/wiki/Integer" title="Integer">integers</a>. </p>
   <p>If 1 is defined as <span class="texhtml"><i>S</i>(0)</span>, then <span class="texhtml"><i>b</i> + 1 = <i>b</i> + <i>S</i>(0) = <i>S</i>(<i>b</i> + 0) = <i>S</i>(<i>b</i>)</span>. That is, <span class="texhtml"><i>b</i> + 1</span> is simply the successor of <span class="texhtml"><i>b</i></span>. </p> 
   <h3><span class="mw-headline" id="Multiplication">Multiplication</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Natural_number&amp;action=edit&amp;section=8" title="Edit section: Multiplication">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
   <p>Analogously, given that addition has been defined, a <a href="/wiki/Multiplication" title="Multiplication">multiplication</a> operator × can be defined via <span class="texhtml"><i>a</i> × 0 = 0</span> and <span class="texhtml"><i>a</i> × S(<i>b</i>) = (<i>a</i> × <i>b</i>) + <i>a</i></span>. This turns <span class="texhtml">(ℕ<sup>*</sup>, ×)</span> into a free commutative monoid with identity element&nbsp;1; a generator set for this monoid is the set of <a href="/wiki/Prime_number" title="Prime number">prime numbers</a>. </p> 
   <h3><span class="mw-headline" id="Relationship_between_addition_and_multiplication">Relationship between addition and multiplication</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Natural_number&amp;action=edit&amp;section=9" title="Edit section: Relationship between addition and multiplication">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
   <p>Addition and multiplication are compatible, which is expressed in the <a href="/wiki/Distributivity" class="mw-redirect" title="Distributivity">distribution law</a>: <span class="texhtml"><i>a</i> × (<i>b</i> + <i>c</i>) = (<i>a</i> × <i>b</i>) + (<i>a</i> × <i>c</i>)</span>. These properties of addition and multiplication make the natural numbers an instance of a <a href="/wiki/Commutative" class="mw-redirect" title="Commutative">commutative</a> <a href="/wiki/Semiring" title="Semiring">semiring</a>. Semirings are an algebraic generalization of the natural numbers where multiplication is not necessarily commutative. The lack of additive inverses, which is equivalent to the fact that <span class="texhtml">ℕ</span> is not <a href="/wiki/Closure_(mathematics)" title="Closure (mathematics)">closed</a> under subtraction (i.e., subtracting one natural from another does not always result in another natural), means that <span class="texhtml">ℕ</span> is <i>not</i> a <a href="/wiki/Ring_(mathematics)" title="Ring (mathematics)">ring</a>; instead it is a <a href="/wiki/Semiring" title="Semiring">semiring</a> (also known as a <i>rig</i>). </p>
   <p>If the natural numbers are taken as "excluding 0", and "starting at 1", the definitions of + and × are as above, except that they begin with <span class="texhtml"><i>a</i> + 1 = <i>S</i>(<i>a</i>)</span> and <span class="texhtml"><i>a</i> × 1 = <i>a</i></span>. </p> 
   <h3><span class="mw-headline" id="Order">Order</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Natural_number&amp;action=edit&amp;section=10" title="Edit section: Order">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
   <p>In this section, juxtaposed variables such as <span class="texhtml"><i>ab</i></span> indicate the product <span class="texhtml"><i>a</i> × <i>b</i></span>, and the standard <a href="/wiki/Order_of_operations" title="Order of operations">order of operations</a> is assumed. </p>
   <p>A <a href="/wiki/Total_order" title="Total order">total order</a> on the natural numbers is defined by letting <span class="texhtml"><i>a</i> ≤ <i>b</i></span> if and only if there exists another natural number <span class="texhtml"><i>c</i></span> where <span class="texhtml"><i>a</i> + <i>c</i> = <i>b</i></span>. This order is compatible with the <a href="/wiki/Arithmetical_operations" class="mw-redirect" title="Arithmetical operations">arithmetical operations</a> in the following sense: if <span class="texhtml"><i>a</i></span>, <span class="texhtml"><i>b</i></span> and <span class="texhtml"><i>c</i></span> are natural numbers and <span class="texhtml"><i>a</i> ≤ <i>b</i></span>, then <span class="texhtml"><i>a</i> + <i>c</i> ≤ <i>b</i> + <i>c</i></span> and <span class="texhtml"><i>ac</i> ≤ <i>bc</i></span>. </p>
   <p>An important property of the natural numbers is that they are <a href="/wiki/Well-order" title="Well-order">well-ordered</a>: every non-empty set of natural numbers has a least element. The rank among well-ordered sets is expressed by an <a href="/wiki/Ordinal_number" title="Ordinal number">ordinal number</a>; for the natural numbers, this is denoted as <a href="/wiki/Omega_(ordinal)" class="mw-redirect" title="Omega (ordinal)"><span class="texhtml"><i>ω</i></span></a> (omega). </p> 
   <h3><span class="mw-headline" id="Division">Division</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Natural_number&amp;action=edit&amp;section=11" title="Edit section: Division">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
   <p>In this section, juxtaposed variables such as <span class="texhtml"><i>ab</i></span> indicate the product <span class="texhtml"><i>a</i> × <i>b</i></span>, and the standard <a href="/wiki/Order_of_operations" title="Order of operations">order of operations</a> is assumed. </p>
   <p>While it is in general not possible to divide one natural number by another and get a natural number as result, the procedure of <i><a href="/wiki/Division_(mathematics)" title="Division (mathematics)">division</a> with remainder</i> is available as a substitute: for any two natural numbers <span class="texhtml"><i>a</i></span> and <span class="texhtml"><i>b</i></span> with <span class="texhtml"><i>b</i> ≠ 0</span> there are natural numbers <span class="texhtml"><i>q</i></span> and <span class="texhtml"><i>r</i></span> such that </p> 
   <dl>
    <dd>
     <span class="texhtml"><i>a</i> = <i>bq</i> + <i>r</i></span> &nbsp;&nbsp;&nbsp;&nbsp; and &nbsp;&nbsp;&nbsp;&nbsp; 
     <span class="texhtml"><i>r</i> &lt; <i>b</i></span>.
    </dd>
   </dl> 
   <p>The number <span class="texhtml"><i>q</i></span> is called the <i><a href="/wiki/Quotient" title="Quotient">quotient</a></i> and <span class="texhtml"><i>r</i></span> is called the <i><a href="/wiki/Remainder" title="Remainder">remainder</a></i> of the division of <span class="texhtml"><i>a</i></span> by&nbsp;<span class="texhtml"><i>b</i></span>. The numbers <span class="texhtml"><i>q</i></span> and <span class="texhtml"><i>r</i></span> are uniquely determined by <span class="texhtml"><i>a</i></span> and&nbsp;<span class="texhtml"><i>b</i></span>. This <a href="/wiki/Euclidean_division" title="Euclidean division">Euclidean division</a> is key to several other properties (<a href="/wiki/Divisibility" class="mw-redirect" title="Divisibility">divisibility</a>), algorithms (such as the <a href="/wiki/Euclidean_algorithm" title="Euclidean algorithm">Euclidean algorithm</a>), and ideas in number theory. </p> 
   <h3><span class="mw-headline" id="Algebraic_properties_satisfied_by_the_natural_numbers">Algebraic properties satisfied by the natural numbers</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Natural_number&amp;action=edit&amp;section=12" title="Edit section: Algebraic properties satisfied by the natural numbers">edit</a><span class="mw-editsection-bracket">]</span></span></h3> 
   <p>The addition (+) and multiplication (×) operations on natural numbers as defined above have several algebraic properties: </p> 
   <ul>
    <li><a href="/wiki/Closure_(mathematics)" title="Closure (mathematics)">Closure</a> under addition and multiplication: for all natural numbers <span class="texhtml"><i>a</i></span> and <span class="texhtml"><i>b</i></span>, both <span class="texhtml"><i>a</i> + <i>b</i></span> and <span class="texhtml"><i>a</i> × <i>b</i></span> are natural numbers.</li> 
    <li><a href="/wiki/Associativity" class="mw-redirect" title="Associativity">Associativity</a>: for all natural numbers <span class="texhtml"><i>a</i></span>, <span class="texhtml"><i>b</i></span>, and <span class="texhtml"><i>c</i></span>, <span class="texhtml"><i>a</i> + (<i>b</i> + <i>c</i>) = (<i>a</i> + <i>b</i>) + <i>c</i></span> and <span class="texhtml"><i>a</i> × (<i>b</i> × <i>c</i>) = (<i>a</i> × <i>b</i>) × <i>c</i></span>.</li> 
    <li><a href="/wiki/Commutativity" class="mw-redirect" title="Commutativity">Commutativity</a>: for all natural numbers <span class="texhtml"><i>a</i></span> and <span class="texhtml"><i>b</i></span>, <span class="texhtml"><i>a</i> + <i>b</i> = <i>b</i> + <i>a</i></span> and <span class="texhtml"><i>a</i> × <i>b</i> = <i>b</i> × <i>a</i></span>.</li> 
    <li>Existence of <a href="/wiki/Identity_element" title="Identity element">identity elements</a>: for every natural number <i>a</i>, <span class="texhtml"><i>a</i> + 0 = <i>a</i></span> and <span class="texhtml"><i>a</i> × 1 = <i>a</i></span>.</li> 
    <li><a href="/wiki/Distributivity" class="mw-redirect" title="Distributivity">Distributivity</a> of multiplication over addition for all natural numbers <span class="texhtml"><i>a</i></span>, <span class="texhtml"><i>b</i></span>, and <span class="texhtml"><i>c</i></span>, <span class="texhtml"><i>a</i> × (<i>b</i> + <i>c</i>) = (<i>a</i> × <i>b</i>) + (<i>a</i> × <i>c</i>)</span>.</li> 
    <li>No nonzero <a href="/wiki/Zero_divisor" title="Zero divisor">zero divisors</a>: if <span class="texhtml"><i>a</i></span> and <span class="texhtml"><i>b</i></span> are natural numbers such that <span class="texhtml"><i>a</i> × <i>b</i> = 0</span>, then <span class="texhtml"><i>a</i> = 0</span> or <span class="texhtml"><i>b</i> = 0</span> (or both).</li>
   </ul> 
   <h2><span class="mw-headline" id="Generalizations">Generalizations</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Natural_number&amp;action=edit&amp;section=13" title="Edit section: Generalizations">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
   <p>Two important generalizations of natural numbers arise from the two uses of counting and ordering: <a href="/wiki/Cardinal_number" title="Cardinal number">cardinal numbers</a> and <a href="/wiki/Ordinal_number" title="Ordinal number">ordinal numbers</a>. </p> 
   <ul>
    <li>A natural number can be used to express the size of a finite set; more precisely, a cardinal number is a measure for the size of a set, which is even suitable for infinite sets. This concept of "size" relies on maps between sets, such that two sets have <a href="/wiki/Equinumerosity" title="Equinumerosity">the same size</a>, exactly if there exists a <a href="/wiki/Bijection" title="Bijection">bijection</a> between them. The set of natural numbers itself, and any bijective image of it, is said to be <i><a href="/wiki/Countable_set" title="Countable set">countably infinite</a></i> and to have <a href="/wiki/Cardinality" title="Cardinality">cardinality</a> <a href="/wiki/Aleph_number#Aleph-null" title="Aleph number">aleph-null</a> (<span class="texhtml">ℵ<sub>0</sub></span>).</li> 
    <li>Natural numbers are also used as <a href="/wiki/Ordinal_numbers_(linguistics)" class="mw-redirect" title="Ordinal numbers (linguistics)">linguistic ordinal numbers</a>: "first", "second", "third", and so forth. This way they can be assigned to the elements of a totally ordered finite set, and also to the elements of any <a href="/wiki/Well-order" title="Well-order">well-ordered</a> countably infinite set. This assignment can be generalized to general well-orderings with a cardinality beyond countability, to yield the ordinal numbers. An ordinal number may also be used to describe the notion of "size" for a well-ordered set, in a sense different from cardinality: if there is an <a href="/wiki/Order_isomorphism" title="Order isomorphism">order isomorphism</a> (more than a bijection!) between two well-ordered sets, they have the same ordinal number. The first ordinal number that is not a natural number is expressed as <span class="texhtml">ω</span>; this is also the ordinal number of the set of natural numbers itself.</li>
   </ul> 
   <p>Many well-ordered sets with cardinal number <span class="texhtml">ℵ<sub>0</sub></span> have an ordinal number greater than <span class="texhtml">ω</span> (the latter is the lowest possible). The least ordinal of cardinality <span class="texhtml">ℵ<sub>0</sub></span> (i.e., the <a href="/wiki/Von_Neumann_cardinal_assignment" title="Von Neumann cardinal assignment">initial ordinal</a>) is <span class="texhtml">ω</span>. </p>
   <p>For <a href="/wiki/Finite_set" title="Finite set">finite</a> well-ordered sets, there is a one-to-one correspondence between ordinal and cardinal numbers; therefore they can both be expressed by the same natural number, the number of elements of the set. This number can also be used to describe the position of an element in a larger finite, or an infinite, <a href="/wiki/Sequence" title="Sequence">sequence</a>. </p>
   <p>A countable <a href="/wiki/Non-standard_model_of_arithmetic" title="Non-standard model of arithmetic">non-standard model of arithmetic</a> satisfying the Peano Arithmetic (i.e., the first-order Peano axioms) was developed by <a href="/wiki/Skolem" class="mw-redirect" title="Skolem">Skolem</a> in 1933. The <a href="/wiki/Hypernatural" class="mw-redirect" title="Hypernatural">hypernatural</a> numbers are an uncountable model that can be constructed from the ordinary natural numbers via the <a href="/wiki/Ultrapower_construction" class="mw-redirect" title="Ultrapower construction">ultrapower construction</a>. </p>
   <p><a href="/wiki/Georges_Reeb" title="Georges Reeb">Georges Reeb</a> used to claim provocatively that <i>The naïve integers don't fill up</i> <span class="texhtml">ℕ</span>. Other generalizations are discussed in the article on <a href="/wiki/Number" title="Number">numbers</a>. </p>

   <h4><span class="mw-headline" id="Von_Neumann_ordinals">Von Neumann ordinals</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Natural_number&amp;action=edit&amp;section=17" title="Edit section: Von Neumann ordinals">edit</a><span class="mw-editsection-bracket">]</span></span></h4> 
   <p>In the area of mathematics called <a href="/wiki/Set_theory" title="Set theory">set theory</a>, a specific construction due to <a href="/wiki/John_von_Neumann" title="John von Neumann">John von Neumann</a><sup id="cite_ref-von_Neumann1923pp199-208_35-0" class="reference"><a href="#cite_note-von_Neumann1923pp199-208-35">[35]</a></sup><sup id="cite_ref-Levy_36-0" class="reference"><a href="#cite_note-Levy-36">[36]</a></sup> defines the natural numbers as follows: </p> 
   <ul>
    <li>Set <span class="texhtml">0 = { }</span>, the <a href="/wiki/Empty_set" title="Empty set">empty set</a>,</li> 
    <li>Define <span class="texhtml"><i>S</i>(<i>a</i>) = <i>a</i> ∪ {<i>a</i>}</span> for every set <span class="texhtml"><i>a</i></span>. <span class="texhtml"><i>S</i>(<i>a</i>)</span> is the successor of <span class="texhtml"><i>a</i></span>, and <span class="texhtml"><i>S</i></span> is called the <a href="/wiki/Successor_function" title="Successor function">successor function</a>.</li> 
    <li>By the <a href="/wiki/Axiom_of_infinity" title="Axiom of infinity">axiom of infinity</a>, there exists a set which contains 0 and is closed under the successor function. Such sets are said to be 'inductive'. The intersection of all such inductive sets is defined to be the set of natural numbers. It can be checked that the set of natural numbers satisfies the <a href="/wiki/Peano_axioms" title="Peano axioms">Peano axioms</a>.</li> 
    <li>It follows that each natural number is equal to the set of all natural numbers less than it:</li>
   </ul> 
   <dl>
    <dd>
     <ul>
      <li><span class="texhtml">0 = { }</span>,</li> 
      <li><span class="texhtml">1 = 0 ∪ {0} = {0} = {{ }}</span>,</li> 
      <li><span class="texhtml">2 = 1 ∪ {1} = {0, 1} = {{ }, {{ }}}</span>,</li> 
      <li><span class="texhtml">3 = 2 ∪ {2} = {0, 1, 2} = {{ }, {{ }}, {{ }, {{ }}}}</span>,</li> 
      <li><span class="texhtml"><i>n</i> = <i>n</i>−1 ∪ {<i>n</i>−1} = {0, 1, …, <i>n</i>−1} = {{ }, {{ }}, …, {{ }, {{ }}, …}}</span>, etc.</li>
     </ul>
    </dd>
   </dl> 
   <p>With this definition, a natural number <span class="texhtml"><i>n</i></span> is a particular set with <span class="texhtml"><i>n</i></span> elements, and <span class="texhtml"><i>n</i> ≤ <i>m</i></span> if and only if <span class="texhtml"><i>n</i></span> is a <a href="/wiki/Subset" title="Subset">subset</a> of <span class="texhtml"><i>m</i></span>. The standard definition, now called definition of <b>von Neumann ordinals</b>, is: "each ordinal is the well-ordered set of all smaller ordinals." </p>
   <p>Also, with this definition, different possible interpretations of notations like <span class="texhtml">ℝ<sup><i>n</i></sup></span> (<span class="texhtml"><i>n</i></span>-tuples versus mappings of <span class="texhtml"><i>n</i></span> into <span class="texhtml">ℝ</span>) coincide. </p>
   <p>Even if one <a href="/wiki/Finitism" title="Finitism">does not accept the axiom of infinity</a> and therefore cannot accept that the set of all natural numbers exists, it is still possible to define any one of these sets. </p> 
   <h4><span class="mw-headline" id="Zermelo_ordinals">Zermelo ordinals</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Natural_number&amp;action=edit&amp;section=18" title="Edit section: Zermelo ordinals">edit</a><span class="mw-editsection-bracket">]</span></span></h4> 
   <p>Although the standard construction is useful, it is not the only possible construction. <a href="/wiki/Ernst_Zermelo" title="Ernst Zermelo">Ernst Zermelo</a>'s construction<sup id="cite_ref-Levy_36-1" class="reference"><a href="#cite_note-Levy-36">[36]</a></sup> goes as follows: </p> 
   <ul>
    <li>Set <span class="texhtml">0 = { }</span></li> 
    <li>Define <span class="texhtml"><i>S</i>(<i>a</i>) = {<i>a</i>}</span>,</li> 
    <li>It then follows that</li>
   </ul> 
   <dl>
    <dd>
     <ul>
      <li><span class="texhtml">0 = { }</span>,</li> 
      <li><span class="texhtml">1 = {0} = {{ }}</span>,</li> 
      <li><span class="texhtml">2 = {1} = {{{ }}}</span>,</li> 
      <li><span class="texhtml"><i>n</i> = {<i>n</i>−1} = {{{…}}}</span>, etc.</li>
     </ul>
    </dd> 
    <dd>
     Each natural number is then equal to the set containing just the natural number preceding it. (This is the definition of 
     <b>Zermelo ordinals</b>.)
    </dd>
   </dl>
