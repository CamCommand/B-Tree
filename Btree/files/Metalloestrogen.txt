  <div class="mw-parser-output">
   <p><b>Metalloestrogens</b> are a class of <a href="/wiki/Inorganic" class="mw-redirect" title="Inorganic">inorganic</a> <a href="/wiki/Xenoestrogen" title="Xenoestrogen">xenoestrogens</a> which can affect the gene expression of human cells responding to <a href="/wiki/Estrogen" title="Estrogen">estrogen</a>. Effects are related to the physiologic function of estrogen because metalloestrogens have shown <a href="/wiki/Affinity_(pharmacology)" class="mw-redirect" title="Affinity (pharmacology)">affinity</a> for <a href="/wiki/Estrogen_receptor" title="Estrogen receptor">estrogen receptors</a>. Because they can mimic estrogen thus activating the receptor, they are considered harmful and potentially linked with <a href="/wiki/Breast_cancer" title="Breast cancer">breast cancer</a>.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup> List of metalloestrogens include <a href="/wiki/Aluminium" title="Aluminium">aluminium</a>, <a href="/wiki/Antimony" title="Antimony">antimony</a>, <a href="/wiki/Arsenite" title="Arsenite">arsenite</a>, <a href="/wiki/Barium" title="Barium">barium</a>, <a href="/wiki/Cadmium" title="Cadmium">cadmium</a>,<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/Chromium" title="Chromium">chromium</a> (Cr(II)), <a href="/wiki/Cobalt" title="Cobalt">cobalt</a>, <a href="/wiki/Copper" title="Copper">copper</a>, <a href="/wiki/Lead" title="Lead">lead</a>, <a href="/wiki/Mercury_(element)" title="Mercury (element)">mercury</a>, <a href="/wiki/Nickel" title="Nickel">nickel</a>, <a href="/wiki/Selenite_(mineral)" title="Selenite (mineral)">selenite</a>, <a href="/wiki/Tin" title="Tin">tin</a> and <a href="/wiki/Vanadate" title="Vanadate">vanadate</a>. </p>
