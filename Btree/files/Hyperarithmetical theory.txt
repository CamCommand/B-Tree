   <p>In <a href="/wiki/Recursion_theory" class="mw-redirect" title="Recursion theory">recursion theory</a>, <b>hyperarithmetic theory</b> is a generalization of <a href="/wiki/Computable_function" title="Computable function">Turing computability</a>. It has close connections with definability in <a href="/wiki/Second-order_arithmetic" title="Second-order arithmetic">second-order arithmetic</a> and with weak systems of <a href="/wiki/Set_theory" title="Set theory">set theory</a> such as <a href="/wiki/Kripke%E2%80%93Platek_set_theory" title="Kripke–Platek set theory">Kripke–Platek set theory</a>. It is an important tool in <a href="/wiki/Effective_descriptive_set_theory" title="Effective descriptive set theory">effective descriptive set theory</a>. </p>
   <p>The central focus of hyperarithmetic theory is the sets of <a href="/wiki/Natural_number" title="Natural number">natural numbers</a> known as <b>hyperarithmetic sets</b>. There are three equivalent ways of defining this class of sets; the study of the relationships between these different definitions is one motivation for the study of hyperarithmetical theory. </p>

A second, equivalent, definition shows that the hyperarithmetical sets can be defined using infinitely iterated <a href="/wiki/Turing_jump" title="Turing jump">Turing jumps</a>. This second definition also shows that the hyperarithmetical sets can be classified into a hierarchy extending the <a href="/wiki/Arithmetical_hierarchy" title="Arithmetical hierarchy">arithmetical hierarchy</a>; the hyperarithmetical sets are exactly the sets that are assigned a rank in this hierarchy. </p>
   <p>Each level of the hyperarithmetical hierarchy corresponds to a countable <a href="/wiki/Ordinal_number" title="Ordinal number">ordinal number</a> (ordinal), but not all countable ordinals correspond to a level of the hierarchy. The ordinals used by the hierarchy are those with an <b><a href="/wiki/Ordinal_notation" title="Ordinal notation">ordinal notation</a></b>, which is a concrete, effective description of the ordinal. </p>
