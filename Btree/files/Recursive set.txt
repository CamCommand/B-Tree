  <div class="mw-parser-output">
   <p>In <a href="/wiki/Computability_theory" title="Computability theory">computability theory</a>, a <a href="/wiki/Set_(mathematics)" title="Set (mathematics)">set</a> of <a href="/wiki/Natural_number" title="Natural number">natural numbers</a> is called <b>recursive</b>, <b>computable</b> or <b>decidable</b> if there is an <a href="/wiki/Algorithm" title="Algorithm">algorithm</a> which takes a number as input, terminates after a finite amount of time (possibly depending on the given number) and correctly decides whether the number belongs to the set or not. </p>
   <p>A more general class of sets consists of the <a href="/wiki/Recursively_enumerable_set" title="Recursively enumerable set">recursively enumerable sets</a>, also called <b>semidecidable</b> sets. For these sets, it is only required that there is an algorithm that correctly decides when a number <i>is</i> in the set; the algorithm may give no answer (but not the wrong answer) for numbers not in the set. </p>
   <p>A set which is not computable is called <b>noncomputable</b> or <b>undecidable</b>. </p>

   <h2><span class="mw-headline" id="Formal_definition">Formal definition</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Recursive_set&amp;action=edit&amp;section=1" title="Edit section: Formal definition">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
   <p>A subset <span class="texhtml"><var>S</var></span> of the <a href="/wiki/Natural_number" title="Natural number">natural numbers</a> is called <b>recursive</b> if there exists a <a href="/wiki/Total_function" class="mw-redirect" title="Total function">total</a> <a href="/wiki/Computable_function" title="Computable function">computable function</a> <span class="texhtml"><i>f</i> </span> such that <span class="texhtml"><i>f</i>(<var>x</var>) = 1 </span> if <span class="texhtml"><var>x</var> ∈ <var>S</var> </span> and <span class="texhtml"><i>f</i>(<var>x</var>) = 0 </span> if <span class="texhtml"><var>x</var> ∉ <var>S</var> </span>. In other words, the set <span class="texhtml"><var>S</var></span> is recursive <a href="/wiki/If_and_only_if" title="If and only if">if and only if</a> the <a href="/wiki/Indicator_function" title="Indicator function">indicator function</a> <span class="texhtml">1<sub><var>S</var></sub></span> is <a href="/wiki/Computable_function" title="Computable function">computable</a>. </p> 
   <h2><span class="mw-headline" id="Examples">Examples</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Recursive_set&amp;action=edit&amp;section=2" title="Edit section: Examples">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
   <ul>
    <li>Every finite or <a href="/wiki/Cofinite" class="mw-redirect" title="Cofinite">cofinite</a> subset of the natural numbers is computable. This includes these special cases: 
     <ul>
      <li>The <a href="/wiki/Empty_set" title="Empty set">empty set</a> is computable.</li> 
      <li>The entire set of natural numbers is computable.</li> 
      <li>Each natural number (<a href="/wiki/Set-theoretic_definition_of_natural_numbers#The_contemporary_standard" title="Set-theoretic definition of natural numbers">as defined in standard set theory</a>) is computable; that is, the set of natural numbers less than a given natural number is computable.</li>
     </ul></li> 
    <li>The set of <a href="/wiki/Prime_number" title="Prime number">prime numbers</a> is computable.</li> 
    <li>A <a href="/wiki/Recursive_language" title="Recursive language">recursive language</a> is a recursive subset of a <a href="/wiki/Formal_language" title="Formal language">formal language</a>.</li> 
    <li>The set of Gödel numbers of arithmetic proofs described in Kurt Gödel's paper "On formally undecidable propositions of Principia Mathematica and related systems I" is computable; see <a href="/wiki/G%C3%B6del%27s_incompleteness_theorems" title="Gödel's incompleteness theorems">Gödel's incompleteness theorems</a>.</li>
   </ul> 
   <h2><span class="mw-headline" id="Properties">Properties</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Recursive_set&amp;action=edit&amp;section=3" title="Edit section: Properties">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
   <p>If <i>A</i> is a recursive set then the <a href="/wiki/Complement_(set_theory)" title="Complement (set theory)">complement</a> of <i>A</i> is a recursive set. If <i>A</i> and <i>B</i> are recursive sets then <i>A</i> ∩ <i>B</i>, <i>A</i> ∪ <i>B</i> and the image of <i>A</i> × <i>B</i> under the <a href="/wiki/Cantor_pairing_function" class="mw-redirect" title="Cantor pairing function">Cantor pairing function</a> are recursive sets. </p>
   <p>A set <i>A</i> is a recursive set <a href="/wiki/If_and_only_if" title="If and only if">if and only if</a> <i>A</i> and the <a href="/wiki/Complement_(set_theory)" title="Complement (set theory)">complement</a> of <i>A</i> are both <a href="/wiki/Recursively_enumerable_set" title="Recursively enumerable set">recursively enumerable sets</a>. The <a href="/wiki/Preimage" class="mw-redirect" title="Preimage">preimage</a> of a recursive set under a <a href="/wiki/Total_function" class="mw-redirect" title="Total function">total</a> <a href="/wiki/Computable_function" title="Computable function">computable function</a> is a recursive set. The image of a computable set under a total computable <a href="/wiki/Bijection" title="Bijection">bijection</a> is computable. </p>
   <p>A set is recursive if and only if it is at level <span class="texhtml"> Δ<span style="display:inline-block;margin-bottom:-0.3em;vertical-align:-0.4em;line-height:1.2em;font-size:80%;text-align:left"><sup style="font-size:inherit;line-height:inherit;vertical-align:baseline">0</sup><br><sub style="font-size:inherit;line-height:inherit;vertical-align:baseline">1</sub></span> </span> of the <a href="/wiki/Arithmetical_hierarchy" title="Arithmetical hierarchy">arithmetical hierarchy</a>. </p>
   <p>A set is recursive if and only if it is either the range of a nondecreasing total computable function or the empty set. The image of a computable set under a nondecreasing total computable function is computable. </p>
