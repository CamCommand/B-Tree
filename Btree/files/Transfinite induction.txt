   <p><b>Transfinite induction</b> is an extension of <a href="/wiki/Mathematical_induction" title="Mathematical induction">mathematical induction</a> to <a href="/wiki/Well-order" title="Well-order">well-ordered sets</a>, for example to sets of <a href="/wiki/Ordinal_number" title="Ordinal number">ordinal numbers</a> or <a href="/wiki/Cardinal_number" title="Cardinal number">cardinal numbers</a>. </p>
   <p>Let P(α) be a <a href="/wiki/Property_(philosophy)#Properties_in_mathematics" title="Property (philosophy)">property</a> defined for all ordinals α. Suppose that whenever P(β) is true for all β &lt; α, then P(α) is also true. Then transfinite induction tells us that P is true for all ordinals. </p>
   <p>Usually the proof is broken down into three cases: </p> 
   <ul>
    <li><b>Zero case:</b> Prove that P(0) is true.</li> 
    <li><b>Successor case:</b> Prove that for any <a href="/wiki/Successor_ordinal" title="Successor ordinal">successor ordinal</a> α+1, P(α+1) follows from P(α) (and, if necessary, P(β) for all β &lt; α).</li> 
    <li><b>Limit case:</b> Prove that for any <a href="/wiki/Limit_ordinal" title="Limit ordinal">limit ordinal</a> λ, P(λ) follows from [P(β) for all β &lt; λ].</li>
   </ul> 
   <p>All three cases are identical except for the type of ordinal considered. They do not formally need to be considered separately, but in practice the proofs are typically so different as to require separate presentations. Zero is sometimes considered a <a href="/wiki/Limit_ordinal" title="Limit ordinal">limit ordinal</a> and then may sometimes be treated in proofs in the same case as limit ordinals. </p>

   <h2><span class="mw-headline" id="Transfinite_recursion">Transfinite recursion</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Transfinite_induction&amp;action=edit&amp;section=1" title="Edit section: Transfinite recursion">edit</a><span class="mw-editsection-bracket">]</span></span></h2>

   <p>More generally, one can define objects by transfinite recursion on any <a href="/wiki/Well-founded_relation" title="Well-founded relation">well-founded relation</a> <i>R</i>. (<i>R</i> need not even be a set; it can be a <a href="/wiki/Proper_class" class="mw-redirect" title="Proper class">proper class</a>, provided it is a <a href="/wiki/Binary_relation#Relations_over_a_set" title="Binary relation">set-like</a> relation; that is, for any <i>x</i>, the collection of all <i>y</i> such that <i>y R x</i> must be a set.) </p> 
   <h2><span class="mw-headline" id="Relationship_to_the_axiom_of_choice">Relationship to the axiom of choice</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Transfinite_induction&amp;action=edit&amp;section=2" title="Edit section: Relationship to the axiom of choice">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
   <p>Proofs or constructions using induction and recursion often use the <a href="/wiki/Axiom_of_choice" title="Axiom of choice">axiom of choice</a> to produce a well-ordered relation that can be treated by transfinite induction. However, if the relation in question is already well-ordered, one can often use transfinite induction without invoking the axiom of choice.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup> For example, many results about <a href="/wiki/Borel_sets" class="mw-redirect" title="Borel sets">Borel sets</a> are proved by transfinite induction on the ordinal rank of the set; these ranks are already well-ordered, so the axiom of choice is not needed to well-order them. </p>

     <a href="/wiki/Real_number" title="Real number">real numbers</a> (this is where the axiom of choice enters via the 
     <a href="/wiki/Well-ordering_theorem" title="Well-ordering theorem">well-ordering theorem</a>), giving a sequence

   <p>The above argument uses the axiom of choice in an essential way at the very beginning, in order to well-order the reals. After that step, the axiom of choice is not used again. </p>
   <p>Other uses of the axiom of choice are more subtle. For example, a construction by transfinite recursion frequently will not specify a <i>unique</i> value for <i>A</i><sub>α+1</sub>, given the sequence up to α, but will specify only a <i>condition</i> that <i>A</i><sub>α+1</sub> must satisfy, and argue that there is at least one set satisfying this condition. If it is not possible to define a unique example of such a set at each stage, then it may be necessary to invoke (some form of) the axiom of choice to select one such at each step. For inductions and recursions of <a href="/wiki/Countable_set" title="Countable set">countable</a> length, the weaker <a href="/wiki/Axiom_of_dependent_choice" title="Axiom of dependent choice">axiom of dependent choice</a> is sufficient. Because there are models of <a href="/wiki/Zermelo%E2%80%93Fraenkel_set_theory" title="Zermelo–Fraenkel set theory">Zermelo–Fraenkel set theory</a> of interest to set theorists that satisfy the axiom of dependent choice but not the full axiom of choice, the knowledge that a particular proof only requires dependent choice can be useful. </p>
