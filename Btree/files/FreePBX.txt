   <p><b>FreePBX</b> is a web-based <a href="/wiki/Open-source_software" title="Open-source software">open-source</a> <a href="/wiki/Graphical_user_interface" title="Graphical user interface">graphical user interface</a> (GUI) that manages <a href="/wiki/Asterisk_(PBX)" title="Asterisk (PBX)">Asterisk</a>, a <a href="/wiki/Voice_over_IP" title="Voice over IP">voice over IP</a> and <a href="/wiki/Telephony" title="Telephony">telephony</a> server.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup> </p>
   <p>FreePBX is licensed under the <a href="/wiki/GNU_General_Public_License" title="GNU General Public License">GNU General Public License</a> version 3.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup> It is a component of the <a href="/wiki/FreePBX_Distro" title="FreePBX Distro">FreePBX Distro</a>, which is an independently maintained <a href="/wiki/Linux" title="Linux">Linux</a> system derived from the source code of the <a href="/wiki/CentOS" title="CentOS">CentOS</a> distribution, having Asterisk pre-installed. It is also included in various third-party distributions such as <i>The FreePBX Distro</i> and <i>AsteriskNow</i>. </p>
   <p>FreePBX was acquired by Schmooze.com in early 2013.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">[3]</a></sup> That firm was, in turn, taken over by <a href="/wiki/Sangoma_Technologies_Corporation" title="Sangoma Technologies Corporation">Sangoma Technologies Corporation</a> on Jan 2, 2015. </p>
   <p>FreePBX is a community of developers and contributors who devote their work to making complicated phone system software easy to use and functional. </p>

   <h2><span class="mw-headline" id="Installation">Installation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=FreePBX&amp;action=edit&amp;section=1" title="Edit section: Installation">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
   <p>FreePBX can be installed as standalone software, or as part of a pre-configured FreePBX Distro that includes the operating system, the Asterisk PBX, FreePBX, and assorted dependencies:<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">[4]</a></sup> </p>
   <p>FreePBX is included in open source distributions such as <a href="/wiki/FreePBX_Distro" title="FreePBX Distro">The Official FreePBX Distro</a>, AsteriskNOW, <a href="/wiki/Elastix" title="Elastix">Elastix</a> and RasPBX </p> 
   <h2><span class="mw-headline" id="Development">Development</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=FreePBX&amp;action=edit&amp;section=2" title="Edit section: Development">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
   <p>The first FreePBX release, version 0.2 (November 28, 2004), was named the Asterisk Management Portal (AMP). The project was renamed to FreePBX for trademark reasons, as <a href="/wiki/Asterisk_(PBX)" title="Asterisk (PBX)">Asterisk</a> is a registered trademark of the <a href="/wiki/Digium" title="Digium">Digium</a> corporation, a subsidiary of <a href="/wiki/Sangoma_Technologies_Corporation" title="Sangoma Technologies Corporation">Sangoma Technologies Corporation</a>. </p>
   <p>New releases of Asterisk have been accommodated by various updates to FreePBX. Updates have included new menus and support for additional capabilities such as voice mail, calling queues, fax, multiple languages, DAHDI and a local user directory. </p>
   <p>Current versions are: </p> 
   <ul>
    <li>FreePBX 2.11 – Completed 2013-05-14- – Adds support for Asterisk 11, Destination popOvers, Module Admin Security Auditing, Chan Motif Module, WebRTC User Control Panel<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">[5]</a></sup></li> 
    <li>FreePBX 12 – STABLE Release 2014-06-23- – Adds support for Asterisk 12&amp;13, New User Control Panel, Module Admin version control, PJSIP Support</li> 
    <li>FreePBX 13 - Stable - adds responsive GUI, support for Asterisk 13, Call Event Logging CEL and reporting, fwconsole CLI system management, Enhanced Bulk User Management, expanded localization support for audio and sound files, and a new global search option.</li> 
    <li>FreePBX 14 - Stable</li> 
    <li>FreePBX 15 - Beta<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">[6]</a></sup></li>
   </ul> 
   <p>FreePBX supports numerous hardware manufacturers, including <a href="/wiki/Aastra_Technologies" title="Aastra Technologies">Aastra Technologies</a>, Algo, AND, <a href="/wiki/AudioCodes" title="AudioCodes">AudioCodes</a>, <a href="/wiki/Cisco_Systems" title="Cisco Systems">Cisco Systems</a>, Cyberdata, <a href="/wiki/Digium" title="Digium">Digium</a>, <a href="/wiki/Grandstream_Networks" title="Grandstream Networks">Grandstream</a>, <a href="/wiki/Mitel" title="Mitel">Mitel</a>, <a href="/wiki/Panasonic" title="Panasonic">Panasonic</a>, <a href="/wiki/Polycom" title="Polycom">Polycom</a>, <a href="/wiki/Sangoma_Technologies_Corporation" title="Sangoma Technologies Corporation">Sangoma</a>, <a href="/wiki/Snom" title="Snom">Snom</a>, <a href="/w/index.php?title=Xorcom&amp;action=edit&amp;redlink=1" class="new" title="Xorcom (page does not exist)">Xorcom</a>, and <a href="/w/index.php?title=Yealink&amp;action=edit&amp;redlink=1" class="new" title="Yealink (page does not exist)">Yealink</a>.<sup id="cite_ref-7" class="reference"><a href="#cite_note-7">[7]</a></sup> FreePBX developers estimate the distro has been deployed in millions of active PBX systems in over 220 countries and territories.<sup id="cite_ref-8" class="reference"><a href="#cite_note-8">[8]</a></sup> </p>
